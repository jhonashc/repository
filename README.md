#### Run docker:

```
docker-compose up -d
```

#### Running the app

```
# install dependencies
pnpm install

# run in dev mode
pnpm dev

# generate production build
pnpm build

# run generated content in dist folder
pnpm start
```

#### Set environmet variables:

```
PORT=
DB_HOST=
DB_NAME=
DB_PASSWORD=
DB_PORT=
DB_USER=
```
